# JewelIce Brand Concept — 음료의 마지막 보석

> "좋은 음료를 완성하는 마지막 재료는 얼음이다.  
> 쥬얼아이스는 그 보석을, 당신의 집으로 가져왔다."

## Brand Concept

| 관점 | 기존 | 새로운 정의 |
|------|------|-----------|
| 카테고리 | 가정용 제빙기 | 음료 완성 도구 |
| 기능 | 얼음을 만든다 | 음료 경험을 설계한다 |
| 포지션 | 편의용품 | 홈바 라이프스타일 |
